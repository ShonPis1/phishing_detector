התקן את הספריות הנדרשות:
pip install pyspellchecker python-bidi

לאחר מכן כל הפקודות מתוך תיקיית code/:
cd code
python main.py - מצב אינטרקטיבי שבו אפשר להקליד בזמן אמת הודעה ולנתח אותה

python main.py -m "" - ניתוח הודעה ישירות משורת ההרצה

python main.py -f .. - ניתוח קובץ טקסט

python main.py --demo - הרצת כל הדוגמאות שנבדקו (30)
